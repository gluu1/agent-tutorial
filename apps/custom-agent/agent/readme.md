# Agent Core 层完整设计

构建完整的 Agent Core 层，包含三层记忆系统、上下文优化、Skills 解析、插件机制和 ReAct 循环。

## 功能模块

| 模块             | 功能                                                             |
| :--------------- | :--------------------------------------------------------------- |
| **三层记忆系统** | 短期记忆（滑动窗口）、会话记忆（摘要提炼）、长期记忆（向量检索） |
| **Skills 系统**  | 解析 Markdown/YAML/JSON 技能文件，动态加载工具                   |
| **上下文优化器** | 优先级组装、Token 预算管理、智能压缩                             |
| **工具注册表**   | 工具注册、执行、确认机制、超时控制                               |
| **Agent 主入口** | 插件装配、组件初始化、生命周期管理                               |
| **Agent Loop**   | ReAct 循环、工具调用处理、流式输出、中断处理                     |
