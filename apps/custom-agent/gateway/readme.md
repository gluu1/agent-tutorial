# Agent 网关层

实现一个支持 WebSocket & SSE 的 RPC 网关，让 Agent 从 CLI 直连升级为网络服务

## 网关层架构

```
┌─────────────────────────────────────────────────────────────────────────┐
│                           Gateway Layer                                 │
├─────────────────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │                      Protocol (帧协议)                          │   │
│  │  - 消息帧格式定义  - 序列化/反序列化  - 消息类型枚举              │   │
│  └─────────────────────────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │                       Server (服务端)                           │   │
│  │  - 连接管理  - 广播机制  - 握手认证  - 路由分发                  │   │
│  └─────────────────────────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │                       Client (客户端)                           │   │
│  │  - 连接建立  - 自动重连  - 心跳保活  - 请求/响应匹配              │   │
│  └─────────────────────────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │                      Handlers (RPC 方法)                        │   │
│  │  - agent.run  - agent.stop  - agent.status  - session.list     │   │
│  └─────────────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────────────┘
```

## 协议流程图

```
Client                                    Gateway
  │                                          │
  │─────── WebSocket Connection ────────────>│
  │                                          │
  │─────── AUTH (token) ────────────────────>│
  │                                          │
  │<────── AUTH_RESPONSE (success) ─────────│
  │                                          │
  │─────── REQUEST (agent.run) ────────────>│
  │                                          │
  │                                          │ ───────> Agent Core
  │                                          │ <─────── Agent Events
  │<────── NOTIFICATION (agent.event) ──────│
  │                                          │
  │<────── RESPONSE (result) ───────────────│
  │                                          │
  │─────── PING ───────────────────────────>│
  │<────── PONG ────────────────────────────│
  │                                          │
```

网关层实现提供了：

- 完整的帧协议：支持请求/响应、通知、流式传输

- WebSocket 服务端：连接管理、认证、路由、心跳

- SSE 服务端：单向流式推送

- 客户端 SDK：自动重连、请求匹配、事件订阅

- RPC 处理器：Agent 运行、停止、状态查询

- 会话管理：用户会话生命周期管理
